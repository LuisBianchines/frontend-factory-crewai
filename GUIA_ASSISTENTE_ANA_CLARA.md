# 🤖 Guia Completo da Assistente Ana Clara

## 🎯 **Quem é a Ana Clara?**

Ana Clara é a assistente especializada do Lapidatto Frontend Factory que te ajuda durante todo o processo de criação de projetos. Ela oferece sugestões contextuais, exemplos práticos e dicas profissionais para cada etapa do processo.

## 🚀 **Como Usar**

### **Ativando a Ajuda**
Durante qualquer pergunta no terminal interativo, digite:
- `ajuda`
- `sugestões`
- `não entendi`
- `help` 
- `?`

### **Respostas Contextuais**
Ana Clara reconhece o tipo de pergunta e oferece ajuda específica:

## 📋 **Guia por Pergunta**

### **1. 📦 Nome do Projeto**
```bash

```

**Ana Clara sugere:**

- Boas práticas para nomes de projeto

```bash

```

**Templates disponíveis:**

- **saas-app**: Aplicação web completa multi-funcional

### **3. 📋 Briefing do Projeto**
```bash

```

**Ana Clara te ajuda com:**

## ✨ **Dicas Pro da Ana Clara**

### **💡 Para Briefings Efetivos:**
- Seja específico sobre o tipo de negócio
- Mencione o problema que está resolvendo
- Inclua referências visuais se tiver
- Defina o tom: profissional, descontraído, etc.

### **💡 Para Definir Público-Alvo:**
- Pense em idade e perfil socioeconômico
- Considere o nível técnico da audiência  
- Defina se é B2B (empresas) ou B2C (consumidores)
- Inclua comportamentos e necessidades específicas

### **💡 Para Escolher Funcionalidades:**
- Liste essenciais primeiro, depois nice-to-have
- Pense na jornada completa do usuário
- Considere integrações com ferramentas externas
- Lembre-se: simplicidade é melhor que complexidade
