# Lapidatto Next.js Base

Template Next.js com App Router e TypeScript utilizado pelos agentes Lapidatto para gerar novos projetos.

## Scripts

- `npm run dev` – inicia o servidor de desenvolvimento Next.js.
- `npm run build` – gera build de produção.
- `npm run start` – inicia servidor em modo produção.
- `npm run lint` – valida o código com o eslint padrão Next.js.

Os arquivos serão sobrescritos pelos agentes Architect, UI/DS e Scaffolder conforme ProjectSpec.
